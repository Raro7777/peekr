01 수집하는 개인정보

슬쩍(peekr)은 사용자의 개인정보를 수집하지 않습니다. 앱에서 사용하는 모든 설정 정보(코인 종목, 위젯 설정 등)는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

02 위치 정보

날씨 기능 사용 시 GPS 위치 정보를 요청할 수 있습니다. 이 정보는 OpenWeatherMap API에 날씨 조회 목적으로만 전달되며, 앱 자체에서 수집·저장하지 않습니다.

03 네트워크 사용

앱은 다음 외부 서비스에만 접속합니다:

  • Binance WebSocket API — 실시간 암호화폐 가격 조회
  • 네이버 금융 API — 국내 주식/지수 가격 조회
  • OpenWeatherMap API — 날씨 정보 조회

이러한 접속은 가격 및 날씨 데이터를 표시하기 위한 목적으로만 이루어지며, 사용자 식별 정보는 전송되지 않습니다.

04 권한 사용

  • 인터넷(INTERNET) — 가격 및 날씨 데이터 수신
  • 위치(ACCESS_FINE_LOCATION) — 날씨 조회용 GPS 좌표 (선택 사항)
  • 다른 앱 위에 표시(SYSTEM_ALERT_WINDOW) — 오버레이 위젯 기능 (선택 사항)
  • 포그라운드 서비스(FOREGROUND_SERVICE) — 오버레이 위젯 백그라운드 실행

05 데이터 저장

사용자의 설정(종목 리스트, 글꼴 크기, 색상 등)은 Android DataStore를 통해 기기 로컬에만 저장됩니다. 앱 삭제 시 모든 데이터가 함께 삭제됩니다.
